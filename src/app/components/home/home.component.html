Bienvenidos a SmartBus

¿Qué es SmartBus?

Es una plataforma digital para la venta de tickets de autobús mediante tarjetas eléctronicas y dispositivos compatibles con NFC, creando un servicio más eficiente y colectando los datos para crear un mejor servicio de transporte público por medio de Machine Learning utilizando algoritmos de agrupación destacando los sitios con mayor frecuencia de uso y generalizando las zonas. Se busca brindar un servicio optimo para todos.
